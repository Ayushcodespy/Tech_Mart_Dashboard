# Grocery E-Commerce Platform

A full-stack grocery delivery platform built with modern technologies.

## Tech Stack

Backend:
- Node.js
- Express
- PostgreSQL
- pg

Admin Panel:
- React
- Tailwind CSS

Customer App:
- React
- Vite
- Mobile-first responsive UI

## Features

User App:
- Product browsing
- Cart system
- Order placement
- User authentication

Admin Panel:
- Product management
- Inventory management
- Order tracking
- Banner management
- Reports by day, month, and date range

Backend:
- REST API
- JWT authentication
- Role based access control
- Configurable store name

## Project Structure

backend/ -> Node.js API server
admin_panel/ -> React admin dashboard
user_app/ -> React customer application

## Deployment

Customer React App: `user_app/`

Backend: Render
Database: Neon PostgreSQL
Admin Panel: Vercel
